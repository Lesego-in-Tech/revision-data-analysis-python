# python functions 
This repository contains Python programs based on Chapter 5: Functions. The files include practical exercises and examples that demonstrate how functions are used to structure programs, improve readability, and make code reusable.

The focus is on understanding both void functions and value-returning functions, as well as working with built-in Python modules such as `math` and `random`.

The programs in this repository are based on the following concepts:
- Introduction to Functions
- Defining and Calling Void Functions
- Designing Programs Using Functions
- Local Variables
- Passing Arguments to Functions
- Global Variables and Global Constants
- Value-Returning Functions
- Generating Random Numbers using `random`
- Writing Your Own Value-Returning Functions
- Using the `math` Module
- Storing Functions in Modules

Through these exercises, the following skills are developed:
- Breaking problems into smaller functional parts
- Writing reusable and structured code using functions
- Understanding scope (local vs global variables)
- Using parameters and return values effectively
- Applying mathematical operations using the `math` module
- Creating simple simulations using random numbers
- Organising code into modules

## 📁 Types of Programs Included
- Function-based calculators
- Programs using random number generation
- Mathematical computation programs
- Programs demonstrating scope of variables
- Modular code examples using multiple functions
